# Test Run Logs

This directory contains timestamped test run reports generated by `run_tests.sh`.

## Log Files

- **test_run_<epoch>.md** - Timestamped test reports in Markdown format
- **latest.md** - Symlink to most recent test run

## Log Format

Each log file contains:

### Header
- Date/time of test run
- Hostname and user
- Test environment details

### Test Execution
- List of test suites run
- Pass/fail status with duration
- Error output for failed tests (expandable details)

### Summary
- Total suites/cases
- Pass/fail counts
- Success rate percentage
- Test coverage breakdown

## Example

```markdown
# pyIRCX Test Run Report

**Date**: 2026-01-14 22:50:00
**Timestamp**: 1736912400
**Host**: bigtux
**User**: jdlewis

---

## Test Environment

- **Python Version**: 3.14.2
- **Test Suites Found**: 8
- **Server**: 127.0.0.1:6667

---

## Test Execution

### Test Results

- ✅ **IRC/IRCX Protocol Tests (115 tests)**: PASSED (45s)
- ✅ **Core Command Tests (28 tests)**: PASSED (12s)
...

---

## Summary

| Metric | Value |
|--------|-------|
| **Total Suites** | 8 |
| **Total Cases** | ~243 |
| **Passed** | 8 |
| **Failed** | 0 |
| **Success Rate** | 100.0% |

### ✅ Result: ALL TESTS PASSED
```

## Viewing Logs

```bash
# View latest test run
cat testing/logs/latest.md

# Or with markdown rendering
glow testing/logs/latest.md

# List all test runs
ls -lh testing/logs/test_run_*.md

# View specific run
cat testing/logs/test_run_1736912400.md
```

## Cleanup

Logs are kept indefinitely. To clean up old logs:

```bash
# Remove logs older than 30 days
find testing/logs -name "test_run_*.md" -mtime +30 -delete

# Keep only last 10 runs
ls -t testing/logs/test_run_*.md | tail -n +11 | xargs rm -f
```

## Integration

Logs are automatically created when running:

```bash
./run_tests.sh
```

Each run creates:
1. Timestamped report: `testing/logs/test_run_<epoch>.md`
2. Updates symlink: `testing/logs/latest.md` → most recent run

## CI/CD Usage

In CI/CD pipelines, upload test reports as artifacts:

```yaml
# GitHub Actions example
- name: Run tests
  run: ./run_tests.sh

- name: Upload test report
  uses: actions/upload-artifact@v3
  with:
    name: test-report
    path: testing/logs/latest.md
```
